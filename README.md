Scale Bot

Description
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
This repository allows violin users to record a simple, one octave scale, and it will return a percent accurarcy. 

Dependencies
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Python dependencies:

  pip install numpy

  pip install scipy

  pip install sounddevice

  pip install fastapi
  
Testing
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
This code was tested on Python 3.7.3 on Windows 32
